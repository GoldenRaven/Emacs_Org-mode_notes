** 自定义横坐标刻度
set xtics 命令直接规定了每个刻度的位置和显示的字符。每一个刻度对应三个参数：
显示字符、刻度位置、刻度等级。刻度等级为 0 时表示主刻度，等级为 1 时表示分刻度。
对于主刻度（等级为 0 时），表示等级的参数也可以省略不写。各个刻度的参数之间用逗号隔开。
如：
#+BEGIN_SRC gnuplot
set xtics ("初一" 0, "初二" 1, "初三" 2)
#+END_SRC
